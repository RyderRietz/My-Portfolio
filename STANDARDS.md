# STANDARDS.md - Technical & Design Specifications

## 1. Tech Stack
- **HTML:** Semantic HTML5 (header, main, section, footer).
- **CSS:** Vanilla CSS3 (Flexbox/Grid).
- **Hosting:** Azure Static Web Apps.
- **CI/CD:** GitHub Actions.

## 2. Design Guidelines
- **Typography:** Professional Sans-Serif (Inter, Roboto, or Arial).
- **Color Palette:**
  - Primary: #CE1141 (Houston Rockets Red for accents).
  - Secondary: #000000 (Black for text/headers).
  - Background: #FFFFFF (White).
- **Accessibility:** All images must have alt tags. Minimum contrast ratio 4.5:1.

## 3. File Structure
- `/index.html`
- `/style.css`
- `/images/` (Contains headshot.jpg)
- `/PRD.md`
- `/STANDARDS.md`
- `/resume.pdf`
