# Product Requirements Document (PRD) - Personal Landing Page

## 1. Purpose
To create a professional "digital home" that showcases my technical skills in Data Mining and Product Management while providing a central hub for recruiters to access my resume and contact information.

## 2. Target Audience
- Recruiters and Hiring Managers
- BAIS Instructors (BAIS:3250, BAIS:3300, BAIS:3500)
- Professional Peers

## 3. Features
- **Hero Section:** Professional headshot and summary.
- **Project Showcase:** Highlights from BAIS:3500 (Data Mining), specifically the Houston Rockets player statistics project.
- **Skills Grid:** Visual representation of Python, SQL, R, and Azure proficiency.
- **Resources:** Direct links to my Resume, LinkedIn, and GitHub repository.

## 4. Success Metrics
- Fully responsive design (Mobile/Desktop).
- 100% deployment success on Azure Static Web Apps via GitHub Actions.
