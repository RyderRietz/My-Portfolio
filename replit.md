# My-Portfolio Landing Page

A simple static HTML landing page portfolio.

## Project Structure

- `index.html` — Main (and only) HTML file, currently displays "Hello World"

## Setup

- Served via Python's built-in HTTP server on port 5000
- Workflow: "Start application" runs `python3 -m http.server 5000 --bind 0.0.0.0`

## Deployment

- Configured as a static site deployment with `publicDir: "."`
